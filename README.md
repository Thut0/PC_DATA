# PC_DATA
SQL analysis project using SSMS to solve 30 business intelligence questions on a global PC sales dataset.

## 📊 Project Overview
This project involves performing exploratory data analysis (EDA) on a global PC sales dataset. The goal was to extract actionable insights regarding revenue, profit margins, and sales performance across different regions and product categories.

## 🛠️ Tools Used
Database Management: SQL Server Management Studio (SSMS).

Language: T-SQL (Transact-SQL).

  Key Techniques:  Data Aggregation (GROUP BY, SUM, AVG).

  Data Type Casting (CAST to bigint for financial accuracy).

  Arithmetic Expressions (Profit and Margin calculations).

  Upcoming: Window Functions (RANK, OVER) and Date Functions (DATEDIFF).
